Напиши коротку програму "Hello World" для main.cpp

Ось проста програма "Hello World" для файлу `main.cpp` на C++:

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```

📌 **Пояснення:**
- `#include <iostream>` — підключає бібліотеку для роботи з введенням/виведенням.
- `std::cout` — виводить текст у консоль.
- `return 0;` — завершує програму з кодом успішного виконання.
