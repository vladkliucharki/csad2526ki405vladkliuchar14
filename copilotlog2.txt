Згенеруй стандартний код для файлу CMakeLists.txt і заповни цей файл цим кодом.

Ось стандартний код для файлу CMakeLists.txt, який підходить для невеликого C++ проекту з використанням генератора Ninja:
cmake_minimum_required(VERSION 3.10)

# Назва проекту та мова
project(MyProject LANGUAGES CXX)

# Встановити стандарт C++
set(CMAKE_CXX_STANDARD 17)
set(CMAKE_CXX_STANDARD_REQUIRED ON)

# Додати головний вихідний файл
add_executable(${PROJECT_NAME} main.cpp)

Що було зроблено:
•	Вказано мінімальну версію CMake.
•	Задано назву проекту та мову програмування.
•	Встановлено стандарт C++17.
•	Додано ціль для створення виконуваного файлу з main.cpp.
Цей шаблон готовий для використання з генератором Ninja у Visual Studio 2022.
